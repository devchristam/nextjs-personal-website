# Next.js personal website
## Introduction
a revamp of my personal website with next.js SSG and tailwind css

## Available Scripts

```bash
# open the dev server
$ npm run dev

# to build the project with next.js ssg
$ npm run build

# to start the next.js server
$ npm run start

$ npm run lint
```

## Using Library
- Next.js
- TailwindCSS
- Front Awesome for SVG icone
